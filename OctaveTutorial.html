Funktionen¶

Eine einfache Funktionsdefinition sieht in Octave so aus:

In [ ]:
function retvar = funcname(arglist)
  body
endfunction

body besteht hier aus einer Folge von Octave statements. Ein einfaches Beispiel ist

In [59]:
function retval = avg (v)
  retval = sum (v) / length (v);
endfunction
In [60]:
x = [1,2,3];
avg(x)

ans =  2

Die return variable kann auch eine Liste sein, und ebenso natrlich die Argumente. Mehr zu Funktionen findet ihr hier.

IF, FOR, WHILE¶

Ein if-statement hat die Form

In [ ]:
if (condition)
  then-body
else
  else-body
endif

Eine for-schleife sieht folgendermaen aus:

In [66]:
for i = 1:5
  i
endfor

disp("\n") %disp ist eine, printfunktion, printf ist aber auch verfgbar

for j = [1,2,5,10]
    j
endfor

i =  1
i =  2
i =  3
i =  4
i =  5


j =  1
j =  2
j =  5
j =  10

...und das while-statement hat die Gestalt

In [ ]:
while (condition)
  body
endwhile

also beispielsweise

In [67]:
i = 1; 
while(i<3)
    i++
endwhile

ans =  2
